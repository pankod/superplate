React Use

react-use has various hooks that you can use with ease.

Hooks are functions that let you ‚Äúhook into‚Äù React state and lifecycle features from function components. Hooks don‚Äôt work inside classes ‚Äî they let you use React without classes.

Refer to official documentation for detailed usage. ‚Üí

react-use#

In this example, we'll use useWindowSize and usePrevious hooks in order to tracks Window dimensions and get the previous state with ease.

import React from "react";import { usePrevious, useWindowSize } from "react-use";
export const ReactUseExample = () => {    const [count, setCount] = React.useState(0);    const prevCount = usePrevious(count);    const { width, height } = useWindowSize();
    return (        <div>            <div>                <p>usePrevious</p>                <p>                    Now: {count}, before: {prevCount}                </p>                <p>                    <button onClick={() => setCount(count + 1)}>                        Increment                    </button>                    <button onClick={() => setCount(count - 1)}>                        Decrement                    </button>                </p>            </div>            <div>                <p>useWindowSize</p>                <p>                    width: {`${width}px`}, height: {`${height}px`}                </p>            </div>        </div>    );};

Adding react-use to your project later#

If you didn't choose the plugin during project creation phase, you can follow the instructions below to add it.

  • Install react-use package.
  • npm
  • yarn
npm install react-use
yarn add react-use
